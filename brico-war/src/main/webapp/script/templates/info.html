Le site La Bonne Aide vous permet de poster, contribuer ou consulter des posts.

Les posts peuvent être soit des questions pratiques concerant toute activité de bricolage, soit une demande d'aide spécifique pour la réalisation de travaux. Dans tous les cas, il sera possible à toute personne inscrite sur le site de contribuer à l'entre-aide entre bricoleurs.

Dans le cas où vous seriez demandeur d'aide, une fois que votre post vous aura apporté satisfaction, vous pourrez alors le valider dans la rubrique Profil pour avertir la communauté qu'une réponse favorable vous a déjà été apportée.

Pour tout renseignement concernant l'utilisation du site ou pour une simple question, vous pouvez écrire à l'adresse mail labonneaide@gmail.com.
